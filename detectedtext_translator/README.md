# LITS
## 이미지 번역 애플리케이션

### 프로그램 목적
AWS RECOKNITION을 활용해 이미지의 텍스트를 추출하고 추출한 텍스트를 GPT-3 TEXT 모델을 활용해 번역을 할 수 있도록 Streamlit으로 웹페이지를 제작함

### 프로그램 툴
1. AWS RECOKNITION
2. GPT-3 TEXT 모델
3. Streamlit
   
### 프로그램 작성 정보
2023/08/11 ~ 2023/08/13 코드 작성
2023/08/14 디버깅 및 에러 수정
